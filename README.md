# Flask Student Management System

A web-based Student Management System developed using Flask and SQLite.

## Features

* Add Student
* View Students
* Delete Student
* Store student data in SQLite database
* Simple web interface using HTML and CSS

## Technologies Used

* Python
* Flask
* SQLite
* HTML
* CSS

## Project Structure

FlaskStudentManagement/

├── app.py

├── students.db

├── static/

│   └── style.css

└── templates/

```
├── index.html

└── students.html
```

## How to Run

1. Install Flask

pip install flask

2. Run the application

python app.py

3. Open in browser

http://127.0.0.1:5000
